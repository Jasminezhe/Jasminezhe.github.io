HEXO博客搭建

1、配置nodeJS

  • 在https://nodejs.org/en/网站下载合适版本的nodeJS；

  • 测试是否安装成功，在终端命令下输入node -v回车，会出现版本信息，则安装成功。

2、安装HEXO

  • 终端命令下输入npm install hexo -g安装HEXO，如果提示错误信息root Administrator,则重新输入sudo npm install hexo -g以管理员方式安装，提示输入密码；

  • 测试是否安装成功，在终端命令下输入hexo,出现以下命令行信息，则安装成功；

  • 在终端下进入桌面，使用mkdir命令创建一个文件夹blog，并进入该文件夹；

  • 输入命令hexo init进行hexo目录的初始化设置；

  • 设置完成后输入命令hexo server启动本地服务器，启动成功后，会提示本地服务器地址，在浏览器中打开后可以本地预览博客。如下图。

3、博客编辑

  • 在终端下进入blog文件夹，输入命令hexo new 文件名，可以新建一个新的博客；

  • 使用MOU软件写入博客内容。

4、博客互联网发布

(1)首先需要github账号，可以在官网注册

(2)配置ssh-key

  • 终端命令下输入ssh-keygen，连续三次回车；

  • 输入vim ~/.ssh/id_rsa.pub，会出现ssh-key的信息，复制下来；

(3)配置github

  • 选择settings进入后在左标签中选择SSH and GPG keys;


  • 选择配置new ssh key，然后在key中粘贴刚刚复制下来的ssh-keygen信息，title随便写。


(4)配置github与本地机器验证

  • 终端下输入git config –global user.name “github用户名”；

  • 终端下输入git config –global user.email “github注册时的邮箱地址”；

没有提示异常就OK啦~~

(5)新建博客域名仓库

  • 选择新建仓库，输入仓库域名，注意，一定要是用户名.github.io,然后选择public，勾选新建readme文档，点击创建；


(6)配置config.yml文件

  • 配置deploy，type: git，repo:域名仓库的https地址，在仓库中点击clone or download 复制仓库域名地址，并粘贴到repo中；


    (7)发布博客

  • 在终端命令中输入npm install hexo-deployer-git –save下载安装配置工具；

  • 可以在本地服务器下预览博客，如果没有问题在终端下输入hexo g将blog中的.md文件编译成css、js等文件；

  • 输入hexo d进行发布，第一次发布会要求输入github的用户名和密码，发布成功后的信息如下。

最后，在浏览器中输入地址，即用户名.github.io就可以在网上查看博客了~~~~

blog说明

  • hexo init命令执行后会配置博客初始化信息，文件夹内容如下：

  • 其中_config.yml存放博客的信息，如设置博客名，替换博客主题等；themes文件夹内存放下载的主题；source/_posts存放新建的博客；


  • 在网址https://github.com/iissnan/hexo-theme-next.git下载主题，选择clone or download复制主题地址，然后在终端命令下输入git clone 地址，等待下载成功；


  • 下载成功后可以在theme文件夹中查看到新下载的主题，然后进入_config.yml把主题名字修改成对应主题的文件夹名就可以了。
